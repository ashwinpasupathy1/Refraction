Help & Wiki

Getting Started

Click Open File to load an Excel spreadsheet (.xlsx) or CSV file. Select a chart type from the left sidebar, configure options in the right panel, then click Generate Plot.

Excel Layout

For bar, box, and violin charts, use a flat header layout: Row 1 = group names, Rows 2+ = numeric values. Each column is one group.

For line and scatter charts, Column 1 = X values, Column 2+ = Y replicates. Row 1 contains the X-axis label and series names.

Statistical Tests

Parametric (default): Uses ANOVA + Tukey HSD for 3+ groups, Welch's t-test for 2 groups. Assumes normally distributed data.

Non-parametric: Uses Kruskal-Wallis + Dunn's post-hoc. Appropriate for non-normal data or ordinal scales.

Paired: Uses paired t-test or Wilcoxon signed-rank. Requires equal group sizes (matched subjects).

Multiple Comparison Corrections

Holm-Bonferroni (recommended): step-down procedure, controls FWER.

Bonferroni: most conservative. BH (FDR): controls false discovery rate — more liberal.

Effect Sizes

Cohen's d: standardized mean difference. d < 0.2 = negligible, 0.2–0.5 = small, 0.5–0.8 = medium, > 0.8 = large.

Hedges' g: bias-corrected version of Cohen's d for small samples.
