Prism Barplot Runner
====================

Requirements (install once):
  pip install matplotlib numpy pandas scipy seaborn openpyxl

To run:
  Double-click PrismBarplot.app  (macOS may ask to allow it first time)
  
  Or from Terminal:
    python3 prism_barplot_app.py

If macOS blocks the app (Gatekeeper):
  Right-click the .app → Open → Open anyway
  Or: xattr -cr PrismBarplot.app

The notebook (prism_barplot_final_updated.ipynb) must stay in the
same folder as the .app, or inside Contents/Resources/ (already there).
